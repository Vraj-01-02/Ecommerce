# MERN E-Commerce Application

A full-stack E-Commerce web application built using the MERN stack, designed with clean architecture and real-world development practices.  
The project covers complete product, cart, order, and user management with separate Admin and Customer applications.

---

## Overview

This project demonstrates how modern e-commerce platforms are developed using scalable frontend architecture, secure backend APIs, and efficient database design.  
It is structured to reflect industry-level project organization and workflows.

---

## Features

### Admin Panel
- Admin authentication
- Product management (add, update, delete)
- Order management
- User overview

### Customer Application
- User authentication
- Product browsing
- Cart management
- Order placement and history

### Backend
- RESTful API architecture
- JWT-based authentication
- Role-based access control
- MongoDB database integration
- Secure image handling using Cloudinary

---
